Personal portfolio for Lydia He, a CS + Game Design student at Georgia Tech with a minor in Industrial Design. Projects span game development (Unity, C#), UI/UX design (Figma), and full-stack development (Java, Python, Firebase). Contact: lhe309@gatech.edu
