Personal portfolio site for Lydia He, a designer and developer studying CS and Game Design at Georgia Tech (minor in Industrial Design). Built with vanilla HTML, CSS, and JavaScript — featuring a Three.js fish aquarium inside a Game Boy shell, a Web Audio generative music system, a custom SVG cursor, and a bento-grid layout with project modals that include case study sections, slideshows, and embedded demos. Projects span game development (Kinetic Vault, Whisker Woods, Current Crisis), UI/UX design (TFT redesign, iTravl), and full-stack development (SpendWise, SpeechWrite).
